# Shopping List

Parts needed for the doughy fermentation controller (excludes the Raspberry Pi 3).

## Required Parts

| Item | Description | Est. Price |
|------|-------------|-----------|
| **DS18B20 waterproof probe** | Stainless steel tip, 1m cable, pre-wired 3-pin (red/yellow/black). The food-safe stainless sheath is important since it sits inside the fermentation container. Search "DS18B20 waterproof stainless steel" on Amazon. | $4-8 |
| **4.7kΩ resistor** | 1/4W through-hole. Pull-up between DS18B20 data line and 3.3V. Often included with the probe; buy a pack if not. | $1 |
| **Single-channel 5V relay module** | Opto-isolated, screw terminals on the high-voltage side. Common boards: SRD-05VDC-SL-C or similar. Must be 5V-logic but triggerable from 3.3V (most are — look for "works with Raspberry Pi"). | $3-5 |
| **Silicone heating mat** | 12V, ~10-20W. Sits under or wraps around the container. Search "12V silicone heater mat" or "reptile heat mat 12V". ~10x15cm works for a 4-6 quart container. 12V is safer than mains-powered. | $8-15 |
| **12V DC power supply** | 2A is plenty for a 20W mat. Standard barrel jack (5.5x2.1mm). | $5-10 |
| **Jumper wires** | Female-to-female for Pi GPIO header connections. A 20-pack of various lengths. | $3-5 |
| **Food-safe container** | Cambro 4qt or 6qt translucent round, with lid. Dough goes inside, heating mat goes under or around it. | $8-12 |
| **MicroSD card** | 8GB+ with Raspberry Pi OS (you may already have one). | $5-10 |

## Total

**~$30-55** excluding the Raspberry Pi 3.
